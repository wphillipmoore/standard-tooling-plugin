
# Release 1.4.9 (2026-04-28)

## Bug fixes

- **strengthen issue-closure step in pr-workflow so agents stop skipping it**
Issue closure was a subsection under After the merge that agents could overlook. Promoted to top-level section, added to the overview flow as step 5, and reinforced the key message throughout: using Ref instead of Fixes defers the timing of closure, not the responsibility. If the PR resolves the issue, the agent must close it explicitly after finalization.
